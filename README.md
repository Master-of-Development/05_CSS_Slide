# 모드 동아리 다섯번째 과제
- 슬라이드 이미지는 3초 이내에 변환되어야 한다.
- 슬라이드 이미지의 전환 순서는 1→2→3→1→2→3→1→2... 와 같은 순서를 유지하여 반복되어야 한다.
- 슬라이드는 무한반복된다.
- 슬라이드는 css로만 만들어야 하면 javascript는 사용할 수 없다.
